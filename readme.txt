MUNXIM APP

A Flutter-based mobile app to help users resell products from platforms like Markaz directly on WhatsApp.

FEATURES:
- Share product images and descriptions
- Lava & Ice themes
- Order delivery system integration (planned)
- Profit tracking (future version)

BUILD:
- Flutter SDK required
- Use 'flutter pub get'
- Build using 'flutter build apk'
